## 🔧 **How to Use the Website**

### **For End Users:**
1. Open `index.html` in a web browser
2. Browse products on the home page
3. Click "View Details" to see product details
4. Navigate to About and Contact pages using the header menu
5. Fill contact form to send a message

### **For Developers:**
1. Add product images to `images/` folder
2. Update products array in `js/script.js` with real data
3. Customize colors in `css/style.css` (root variables)
4. Add background image as `gifts-bg.jpg`
5. Update About and Contact page content
6. Test on all screen sizes
7. Deploy to web server
